# Sounds

Place calming ambient audio files here as MP3.

Expected files for `CalmingSoundsScreen`:
- `rain.mp3`
- `ocean.mp3`
- `forest.mp3`
- `white_noise.mp3`
- `stream.mp3`

The UI works without these files; playback shows a snackbar if an asset is missing.

Referenced in `pubspec.yaml` under `assets/sounds/`.
